This repository contains implementations of various Machine Learning and AI models applied to real-world datasets. It includes end-to-end workflows such as:

Data preprocessing
Exploratory Data Analysis (EDA)
Model training & evaluation
Performance comparison
Visualization of results

The primary goal of this repository is to provide a flexible and interactive environment for experimenting with different ML algorithms and analyzing their performance.
